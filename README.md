## readaloud
A KOReader plugin that plays **EPUB3 Media Overlay** audio while highlighting the
currently spoken text element in sync.  Targets books produced by the
[Storyteller platform](https://gitlab.com/storyteller-platform/storyteller).

## Heads up
This was built with AI, so yeah AI slop so expect some rough edges. I got it built because readaloud (media overlay) was the one feature I was missing in KOReader and I couldn't find an existing plugin for it.

### Tested on
The files tested are all from Storyteller, no other epubs where tested.
Only tested on Android, since that's all I have:
- Samsung Galaxy S25
- Boox Go Color 7 Gen II

It mostly works, with a few quirks here and there.
If you can take this and turn into a proper plugin, please do so.
